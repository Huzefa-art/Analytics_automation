- [ ] NOTE: SLACK_BOT_TOKEN in .env starts with xapp- (app-level token). The
      Bot Token CSV upload path requires an xoxb- bot token. Update the token
      in .env or the Slack app settings if file uploads are needed.
      HOW TO GET THE RIGHT TOKEN:
        1. Go to api.slack.com/apps → select your app
        2. Click "OAuth & Permissions" in the left sidebar
        3. Under "Bot Token Scopes" make sure these are added:
           - files:write  (to upload files)
           - channels:read  (to resolve channel names to IDs)
           - chat:write  (to post messages)
        4. Click "Install to Workspace" (or Reinstall)
        5. Copy the "Bot User OAuth Token" — it starts with xoxb-
        6. Paste it in the Slack modal or update SLACK_BOT_TOKEN in .env
